Omdat vs. want — het verschil
The difference between omdat and want
▼
OMDAT → bijzin → werkwoord achteraan: "Ik kan niet komen omdat ik ziek ben." "Ik schrijf u omdat ik een probleem heb." WANT → hoofdzin → werkwoord op normale plek: "Ik kan niet komen, want ik ben ziek." "Dit is vervelend, want ik moet morgen werken." DUS → gevolg → gewone volgorde: "Ik was ziek, dus ik kon niet komen."

De dt-regel in één minuut
The dt-rule in one minute
▼
Stam van het werkwoord = de hij-vorm minus -t: "hij werkt" → stam = "werk" Tegenwoordige tijd: ik werk (geen t) jij werkt (met t, tenzij inversie: "werk jij") hij/zij werkt (met t) Verleden tijd: Zacht klinker aan einde stam (t, k, f, s, ch, p = 't kofschip): → -te / -ten: "ik werkte, wij werkten" Anders: → -de / -den: "ik leefde" — wacht, 'f' is in 't kofschip → "ik leefde"? Nee: stam van leven = "leef" → 'f' in kofschip → "leefde" ✓ Makkelijkste test: vervang het werkwoord door "werken": "hij werkt" → "hij word-t" → "hij wordt" ✓ "gewerkt / geword-t" → gebruik de t-versie als stam eindigt op t
